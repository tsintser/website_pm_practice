Project Overview

This is an educational IT project aimed at practicing Project Management skills.

The goal of the project is to design and deliver a landing page that allows users to submit a lead request (phone number only) and complete an online payment directly on the page.

The project simulates a real business scenario, including requirements gathering, planning, risk assessment, and delivery.

Role: Project Manager
Scope: Full project lifecycle (from idea to delivery)
Project type: Educational / Practice project
