# website_pm_practice
This is an educational project created to practice IT Project Management skills.

Role: Project Manager  
Scope: Full project lifecycle from idea to delivery  
Tools: Notion, GitHub  

The repository contains:
- project goals and vision
- requirements and user stories
- roadmap and planning
- risks and retrospective

# Project Overview
This project simulates a real-world business scenario and is designed to demonstrate Project Management skills in a digital product environment.

The objective of the project is to plan and deliver a landing page that enables users to:
- submit a lead request using a phone number only
- complete an online payment directly on the page

The project focuses on translating business goals into structured requirements, aligning stakeholders, managing risks, and delivering a user-centric solution.

**Role:** Project Manager  
**Responsibilities:**  
- Defining project scope and objectives  
- Stakeholder alignment  
- Requirements documentation  
- Planning and delivery coordination  

**Project Type:** Educational / Practice Project  
**Industry:** Digital Products / Online Services  

# Project Goals
- Design a landing page focused on lead generation and conversion
- Minimize friction in the user journey by collecting only essential data
- Enable seamless online payment within the landing page
- Ensure clarity and transparency of the value proposition

# Success Criteria
- User can submit a lead request using a phone number only
- User can complete a payment without leaving the landing page
- All key user scenarios are documented and validated
- Project scope, risks, and assumptions are clearly defined
- Documentation is structured and accessible
